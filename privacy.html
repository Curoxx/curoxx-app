Curoxx

Política de Privacidade

Última atualização: junho de 2026

1. Introdução

O Curoxx é um aplicativo de menu bar para macOS desenvolvido por mafixx. Esta política de privacidade descreve como o aplicativo coleta, usa e protege suas informações.

2. Dados coletados

O Curoxx não coleta nenhum dado pessoal do usuário. O aplicativo não requer criação de conta, login ou qualquer identificação.

As únicas informações armazenadas localmente no seu dispositivo são:

  • Histórico de cotações (armazenado apenas no seu Mac, no arquivo curoxx_history.json)
  • Preferências do aplicativo (moeda base, intervalo de atualização, número de moedas visíveis)

Nenhuma dessas informações é enviada para servidores externos ou compartilhada com terceiros.

3. Acesso à internet

O Curoxx acessa a internet exclusivamente para buscar cotações de câmbio em tempo real. As requisições são feitas para a API pública da AwesomeAPI (economia.awesomeapi.com.br). Nenhum dado do usuário é enviado nessas requisições.

4. Dados de terceiros

O aplicativo utiliza a API pública da AwesomeAPI para obter cotações de câmbio. Consulte a política de privacidade da AwesomeAPI para informações sobre como eles tratam os dados de acesso à API.

5. Compartilhamento de dados

O Curoxx não compartilha nenhum dado com terceiros. Não utilizamos serviços de análise, rastreamento, publicidade ou qualquer outro serviço que colete dados dos usuários.

6. Segurança

Todos os dados armazenados pelo Curoxx ficam exclusivamente no seu dispositivo. O aplicativo opera dentro do sandbox da App Store da Apple, que fornece camadas adicionais de segurança e proteção.

7. Crianças

O Curoxx não é direcionado a crianças menores de 13 anos e não coleta intencionalmente informações de crianças.

8. Alterações nesta política

Podemos atualizar esta política de privacidade periodicamente. Recomendamos verificar esta página ocasionalmente para se manter informado sobre eventuais mudanças.

9. Contato

Se você tiver dúvidas sobre esta política de privacidade, entre em contato pelo e-mail: curoxx@proton.me
